## Objetivo

Este projeto tem como objetivo fortalecer os fundamentos da programação através da resolução de exercícios envolvendo:

- Entrada, processamento e saída de dados
- Variáveis
- Operadores matemáticos
- Cálculos geométricos
- Lógica de programação
- Estruturas básicas da linguagem Python

## Exercícios Desenvolvidos

### Geometria

- Área do Círculo
- Área do Quadrado
- Área do Retângulo
- Área do Triângulo
- Área do Paralelogramo
- Área do Losango
- Área do Pentágono
- Área do Hexágono

### Operações Básicas

- Antecessor
- Sucessor

## Tecnologias Utilizadas

- Python 3
